🎯 COMO USAR:

  PASSO 1: Na Página Inicial
    → Escolha Área (Matemática, Programação, Inglês)
    → Escolha Dificuldade (Fácil, Médio, Difícil)
    → ⭐ Escolha Modo:
       • ✅ Respostas Fechadas (para testar)
       • 💡 Modo Guiado (para aprender)
    → Clique [INICIAR →]

  PASSO 2: Durante o Quiz
    MODO FECHADO:
      - Vê 4 opções sem dicas
      - Escolhe uma
      - Recebe feedback

    MODO GUIADO:
      - Vê 4 opções, 1 destacada em verde
      - Sabe qual é a resposta
      - Escolhe para confirmar
      - Recebe feedback

  PASSO 3: No Resultado
    - Ver modo utilizado
    - Ver pontuação (igual em ambos)
    - Receber sugestões

📈 PONTUAÇÃO:

  MODO FECHADO:
    Acerto:   +10 pontos base + bônus tempo (~18 total)
    Erro:     0 pontos
    Streak:   +1 para acerto, reset para erro
    XP:       +15 para acerto, +2 para erro

  MODO GUIADO:
    Acerto:   +10 pontos base + bônus tempo (~18 total)
    Erro:     0 pontos (mesmo vendo a resposta!)
    Streak:   +1 para acerto, reset para erro
    XP:       +15 para acerto, +2 para erro

  → Pontuação IGUAL em ambos modos
  → Incentiva aprender CORRETAMENTE, não apenas ganhar pontos
