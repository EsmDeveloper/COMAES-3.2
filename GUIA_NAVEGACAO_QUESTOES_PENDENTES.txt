AÇÕES DISPONÍVEIS:

Para cada questão pendente você pode:

1. 👁️ VISUALIZAR DETALHES
   • Clique no título da questão
   • Vê: Descrição, tipo, dificuldade, opções, resposta correta

2. ✅ APROVAR
   • Clique botão "Aprovar"
   • Questão vai para "Questões dos Colaboradores"
   • Pode ser usada em torneios/testes

3. ❌ REJEITAR
   • Clique botão "Rejeitar"
   • Questão é deletada ou movida para rejeitadas
   • Colaborador é notificado

4. 🔍 BUSCAR
   • Use campo de busca no topo
   • Busca por: Título, Colaborador, ID

█ POSSÍVEIS PROBLEMAS & SOLUÇÕES

PROBLEMA 1: Aba "Questões Pendentes" não aparece no menu
SOLUÇÃO:
  1. Verifique se você está logado como admin
  2. Atualize a página (Ctrl+R ou Cmd+R)
  3. Limpe cache do navegador (Ctrl+Shift+Delete)
  4. Faça login novamente

PROBLEMA 2: Aba carrega mas fica vazia (nenhuma questão)
CAUSA: Não há questões pendentes de aprovação
AÇÃO: Isso é normal se nenhum colaborador submeteu questões
PRÓXIMO PASSO: Crie questões como colaborador e depois aprove

PROBLEMA 3: Aba carrega mas depois desaparece/quebra
CAUSA: ⚠️ BUG CONHECIDO identific ado nesta auditoria
SOLUÇÃO TEMPORÁRIA:
  1. Atualize a página (F5)
  2. Volte para menu principal e clique novamente
  3. O bug está em vias de ser corrigido

PROBLEMA 4: "Você não tem permissão para acessar"
CAUSA: Você não é admin
SOLUÇÃO:
  1. Faça logout
  2. Login como usuário com role "admin"
  3. Verifique no banco: SELECT role FROM usuarios WHERE email='seu@email.com';

█ ESTRUTURA DE CÓDIGO (Para Devs)

Como o painel admin é estruturado:

FrontEnd/src/Administrador/AdminDashboard.jsx
  ├─ Importa: QuestoesPendentesTab
  ├─ menuSections define as abas
  └─ activeTab controla qual aba é mostrada

Quando você clica em "Questões Pendentes":
  1. activeTab muda para 'questoes-pendentes'
  2. AdminDashboard renderiza: <QuestoesPendentesTab />
  3. O componente carrega questões do endpoint:
     GET /api/questoes/pendentes

O componente relevante:
  FrontEnd/src/Administrador/QuestoesPendentesTab.jsx

█ PRÓXIMOS PASSOS

Depois de chegar na aba "Questões Pendentes", você pode:

1. APROVAR QUESTÃO
   → Clique [Aprovar]
   → Questão vai para "Questões dos Colaboradores"

2. REJEITAR QUESTÃO
   → Clique [Rejeitar]
   → Insira motivo (opcional)
   → Colaborador é notificado

3. VISUALIZAR DETALHES
   → Clique no título
   → Vê tudo: descrição, opções, resposta correta, etc

4. BUSCAR QUESTÃO
   → Use campo de busca
   → Digite: título, autor, ID

5. CRIAR NOVO BLOCO COM ESSAS QUESTÕES
   → Aprove a questão
   → Vá para "Questões dos Colaboradores"
   → Organize em blocos
   → Adicione a torneios/testes
