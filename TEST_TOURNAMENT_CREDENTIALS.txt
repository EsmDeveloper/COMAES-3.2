🛠️ TESTING COMMANDS
───────────────────────────────────────────────────────────────────────────────

View Tournament in Database:
  mysql> SELECT id, titulo, tipo_torneio, disciplina_especifica, status 
         FROM torneios WHERE id = 61;

View Associated Blocks:
  mysql> SELECT * FROM torneio_blocos WHERE torneio_id = 61;

View Questions in Tournament:
  mysql> SELECT bqi.questao_id, qtc.enunciado
         FROM bloco_questoes_items bqi
         JOIN torneio_blocos tb ON bqi.bloco_id = tb.bloco_id
         JOIN questoes_teste_conhecimento qtc ON bqi.questao_id = qtc.id
         WHERE tb.torneio_id = 61
         ORDER BY bqi.ordem;

Count Questions:
  mysql> SELECT COUNT(*) as total_questoes FROM bloco_questoes_items 
         WHERE bloco_id IN (1, 2);
