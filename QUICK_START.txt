🚀 COMO TESTAR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Compilar:
   npm run build

2. Abrir painel administrativo:
   Menu > Questões & Conteúdo > Questões Pendentes

3. Você verá:
   ✓ Caixa azul com guia de fluxo
   ✓ Lista de questões pendentes
   ✓ Botões de ação

4. Aprovar uma questão:
   ✓ Clique "Ver detalhes"
   ✓ Revise a questão
   ✓ Clique "✅ APROVAR"
   ✓ Veja o toast de sucesso (canto inferior direito)

5. Ir para "Questões dos Colaboradores":
   Menu > Questões & Conteúdo > Questões dos Colaboradores

6. Questão aprovada aparece:
   ✓ Clique para expandir
   ✓ Veja novos botões: 🏆 Torneio, 📚 Teste
   ✓ Role até final para ver guia visual


✨ NOVIDADES PRINCIPAIS

📋 GUIA DE FLUXO (QuestoesPendentesTab)
   Fluxo de aprovação:
   1. Colaborador cria questão
   2. Você aprova AQUI
   3. Questão vai para "Questões dos Colaboradores"
   4. Admin pode adicionar a Torneios ou Testes

🎉 TOAST NOTIFICATION (Feedback ao aprovar)
   ✓ Questão aprovada!
     Ela agora está disponível em "Questões dos Colaboradores"
     e pode ser adicionada a Torneios ou Testes
   (Desaparece automaticamente em 5s)

🎬 BOTÕES DE AÇÃO (QuestoesColaboradoresTab)
   [✏️ Editar]
   [🏆 Adicionar a Torneio]
   [📚 Adicionar a Teste]
   [👤 Ver Autor]

📘 GUIA VISUAL (Final da página)
   Como adicionar a Torneios/Testes?

   🏆 PARA TORNEIOS:
      1. Clique "Adicionar a Torneio"
      2. Selecione ou crie um Bloco
      3. Associe ao Torneio
      4. Pronto no quiz!

   📚 PARA TESTES:
      1. Clique "Adicionar a Teste"
      2. Escolha categoria
      3. Configure dificuldade
      4. Pronto nos testes!
