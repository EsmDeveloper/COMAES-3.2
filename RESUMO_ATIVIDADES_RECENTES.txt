🐛 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

❌ Problema: Ainda vejo "João Silva", "Maria Santos"
   → Causa: Backend não reiniciado
   → Solução: Ctrl+C para parar + npm start para iniciar

❌ Problema: Erro 500
   → Causa: Possível erro na query
   → Solução: Verificar logs do backend para mensagem exata

❌ Problema: Nenhuma atividade aparece
   → Causa: Nenhuma atividade nas últimas 24h no BD
   → Solução: Crie dados de teste (inscrição, teste, etc)

❌ Problema: "Cannot read property 'nome'"
   → Causa: Associações Sequelize não configuradas
   → Solução: Verificar BackEnd/models/associations.js
