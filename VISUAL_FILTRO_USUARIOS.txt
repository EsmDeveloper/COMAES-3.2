╔════════════════════════════════════════════════════════════════════════════════╗
║                              TIPO DE USUÁRIO                                    ║
╚════════════════════════════════════════════════════════════════════════════════╝

ESTUDANTE (Usuário Regular)
├─ role = 'estudante'
├─ isAdmin = false
├─ Pode participar de torneios
├─ Pode resolver questões
└─ Tem perfil pessoal

PROFESSOR / COLABORADOR
├─ role = 'colaborador'
├─ disciplina_colaborador preenchida
├─ Pode criar questões para sua disciplina
├─ Pode gerenciar suas questões
└─ Pode visualizar desempenho de estudantes

ADMINISTRADOR
├─ isAdmin = true
├─ Ou role = 'admin'
├─ Acesso completo ao painel administrativo
├─ Pode gerenciar usuários, torneios, questões
├─ Pode visualizar estatísticas do sistema
└─ Pode configurar parâmetros da plataforma

╔════════════════════════════════════════════════════════════════════════════════╗
║                            EXEMPLOS DE FILTRAGEM                               ║
╚════════════════════════════════════════════════════════════════════════════════╝

CENÁRIO 1: Listar todos os estudantes
→ Selecionar "Estudante" no dropdown
→ Exibe apenas usuários com role = 'estudante' e isAdmin = false

CENÁRIO 2: Encontrar professor de Programação
→ Selecionar "Professor/Colaborador"
→ Digitar "Programação" na busca
→ Exibe apenas colaboradores com "Programação" no nome ou disciplina

CENÁRIO 3: Gerenciar todos os administradores
→ Selecionar "Administrador"
→ Exibe todos os usuários com isAdmin = true ou role = 'admin'
→ Admin master (id=1) não é visível para admins secundários

CENÁRIO 4: Busca geral com filtro
→ Selecionar tipo (ex: "Estudante")
→ Digitar termo (ex: "João")
→ Exibe estudantes com "João" no nome ou email
