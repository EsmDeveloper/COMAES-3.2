💡 QUICK TROUBLESHOOTING
──────────────────────────────────────────────────────────────────────

Issue: 401 Unauthorized
→ Check Authorization header and token validity

Issue: 403 Você só pode criar...
→ User's discipline doesn't match content's discipline

Issue: 404 Not found
→ Content belongs to different user or doesn't exist

Issue: 500 Server error
→ Run: node setup-colaborador-workflow.js

Issue: Database errors
→ Verify MySQL running on localhost:3306
