  Ficheiro Principal: FrontEnd/src/Paginas/Primarias/AuthContainer.jsx

  ✅ Import WaitingScreen (linha 11)
  ✅ Estados para controlar (linhas 23-25)
  ✅ Logout adicionado ao useAuth (linha 15)
  ✅ Check no handleLoginSubmit (linhas 156-161)
  ✅ Renderização condicional (linhas 373-390)

  LÓGICA:
    if (body.data.role === 'colaborador' && body.data.status_colaborador === 'pendente')
      → setShowWaitingScreen(true)
    else
      → navigate(getPostLoginRoute(body.data))

  CALLBACKS:
    onApproved:
      • setShowWaitingScreen(false)
      • navigate(getPostLoginRoute(user))  // /colaborador/dashboard
    
    onRejected:
      • setShowWaitingScreen(false)
      • logout() - limpar session
      • setMode('login') - voltar para login

FLUXO COMPLETO:
  1. Colaborador faz login com status_colaborador = 'pendente'
  2. AuthContainer detecta e mostra WaitingScreen
  3. WaitingScreen verifica status a cada 5 segundos
  4. Admin aprova → status muda para 'aprovado'
  5. WaitingScreen detecta → mostra sucesso
  6. Após 2s → auto-redireciona para /colaborador/dashboard

VERIFICAÇÃO:
