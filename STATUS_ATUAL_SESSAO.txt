🎯 TAREFA CONCLUÍDA: Corrigir Contagem de Questões no Torneio

┌──────────────────────────────────────────────────────────────────────────┐
│ PROBLEMA ORIGINAL                                                        │
└──────────────────────────────────────────────────────────────────────────┘

Na aba de Torneios do painel admin, o card mostrava:
  ❌ "14 blocos · 0 questões"

Mesmo que os blocos tivessem questões dentro, a contagem era sempre "0".

┌──────────────────────────────────────────────────────────────────────────┐
│ ROOT CAUSE IDENTIFICADA                                                  │
└──────────────────────────────────────────────────────────────────────────┘

Backend:
  • Endpoint GET /api/torneios/{id}/blocos retornava apenas total_questoes (int)
  • NÃO incluía o array 'questoes' com os dados reais
  • Frontend não conseguia expandir e mostrar questões

Frontend:
  • Card tentava renderizar questões do estado vazio
  • handleToggleExpand tentava carregar via GET /api/blocos/{id}
  • Mas isso não atualizava o card (questões já tinham sido contadas)

┌──────────────────────────────────────────────────────────────────────────┐
│ SOLUÇÃO IMPLEMENTADA                                                     │
└──────────────────────────────────────────────────────────────────────────┘

✅ BACKEND (BlocosController.js)
   Modificar listarBlocosDoTorneio():
   • Carregar BlocoQuestaoItem com include de QuestaoTesteConhecimento
   • Normalizar dados de questões (opcoes JSON parsing)
   • Retornar array 'questoes' junto com total_questoes
   
   Antes: { id, titulo, total_questoes: 14 }
   Depois: { id, titulo, total_questoes: 14, questoes: [...14 items] }

✅ FRONTEND (BlocoQuestoesManager.jsx)
   Adicionar useEffect em BlocoCard:
   • Detectar se bloco já vem com questões
   • Carregar automaticamente no estado local
   • Sem esperar usuário expandir
   
   Otimizar handleToggleExpand():
   • Verificar se questões já estão no bloco
   • Se sim, usar do estado (sem requisição)
   • Se não, fazer fallback para requisição antiga
