API ENDPOINTS
=============

1. GET /api/questoes/admin/pendentes
   - Lists all pending questions
   - Supports pagination: ?pagina=1&limite=20
   - Optional filters: ?disciplina=matematica&dificuldade=facil
   - Access: Admin only

2. PUT /api/questoes/:id/aprovar
   - Approves a pending question
   - No request body required
   - Returns updated question with review fields
   - Access: Admin only

3. PUT /api/questoes/:id/rejeitar
   - Rejects a question with reason
   - Request body: { "motivo_rejeicao": "reason text" }
   - Returns updated question with all fields
   - Access: Admin only
