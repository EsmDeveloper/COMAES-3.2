📍 ONDE ESTÁ O ÍCONE?
═════════════════════════════════════════════════════════════════════════════════

No painel do colaborador, canto SUPERIOR DIREITO do header.

    🔔 ← Aqui!  (com badge "5" se tiver notificações)


❓ O QUE FAZE O ÍCONE?
═════════════════════════════════════════════════════════════════════════════════

1. Clique no ícone 🔔
2. Abre um menu com suas notificações
3. Cada notificação mostra:
   - O tipo (Torneio, Resultado, etc.)
   - O título e mensagem
   - Quanto tempo atrás foi enviada

4. Clique em uma notificação para marcá-la como lida
5. Use o botão para marcar todas de uma vez

🚀 COMO USAR COMO COLABORADOR?
═════════════════════════════════════════════════════════════════════════════════

1. Abra o painel do colaborador
2. Veja o número de notificações no badge (ex: 🔔(5))
3. Clique no ícone para abrir
4. Leia suas notificações
5. Clique em uma para marcar como lida
6. Ou use "Marcar todas como lidas"


🔧 COMO USAR COMO ADMIN (ENVIAR)?
═════════════════════════════════════════════════════════════════════════════════

1. Vá para Admin Panel
2. Clique em "Notificações"
3. Selecione os colaboradores
4. Escolha o tipo (Torneio, Aprovação, etc.)
5. Escreva o título e mensagem
6. Clique em "Enviar"

As notificações aparecerão nos painéis dos colaboradores!

🆘 NÃO ESTÁ FUNCIONANDO?
═════════════════════════════════════════════════════════════════════════════════

1. Verifique se está logado como colaborador
2. Procure o ícone 🔔 no topo direito
3. Abra DevTools (F12) → Console
4. Procure por erros em vermelho
5. Envie uma notificação do Admin para testar

Se ainda não funcionar:
  - Verifique se a API está respondendo: /api/notificacoes/usuario/{id}
  - Veja se há dados de usuário corretos


📊 TECNOLOGIA USADA
═════════════════════════════════════════════════════════════════════════════════

React Hooks:
  - useState: Para gerenciar estados
  - useEffect: Para polling automático
  - useCallback: Para otimizar funções
  - useRef: Para manage do intervalo

Bibliotecas:
  - lucide-react: Ícone Bell
  - framer-motion: Animações do modal

Integração:
  - Fetch API para chamadas HTTP
  - LocalStorage para token de autenticação
