Validações em tempo real\n└─ Auto-save de rascunho\n\nPasso 2: Configuração\n├─ Datas, Limites, Toggles\n├─ Validações (data final > data inicial)\n└─ Resumo visual\n\nPasso 3: Questões\n├─ Criar, Página Unificada (6 Abas)\n```\nAba 1: Visão Geral\n├─ Dashboard com métricas\n└─ Botões de ação\n\nAba 2: Questões\n├─ Gerenciar questões\n└─ Integração com CreateQuestaoForm\n\nAba 3: Participantes\n├─ Validar e gerenciar\n└─ Filtros e busca\n\nAba 4: Ranking\n├─ Visualizar em tempo real\n└─ Exportação\n\nAba 5: Estatísticas\n├─ Análises e gráficos\n└─ Relatórios\n\nAba 6: Configurações\n├─ Editar dados\n└─ Histórico de alterações\n```\n\n### RF3: Integração Automática\n```\n✓ torneio_id atribuído automaticamente\n✓ Questões aparecem imediatamente\n✓ Sem seleção manual\n✓ Utiliza exclusivamente Questao.js\n```\n\n### RF4: Melhorias de UX\n```\n✓ Barra de progresso\n✓ Auto-save de rascunho\n✓ Validação instantânea\n✓ Mensagens claras de erro\n✓ Confirmações visuais\n✓ Loading states\n✓ Navegação entre passos\n```\n\n### RF5: Validações de Negócio\n```\n✗ Data final < data inicial\n✗ Torneio sem nome\n✗ Torneio sem disciplina\n✗ Torneio publicado sem questões\n✗ Limite de participantes inválido\n```\n\n---\n\n## 🏗️ Arquitetura em 1 Página\n\n### Frontend\n```\nTournamentWizard/\n├─ TournamentWizard.jsx (componente principal)\n├─ Página\n\n### Fluxo 1: Admin → Torneio → Questões\n```\nAdmin cria torneio\n    ↓\nSistema retorna torneio_id\n    ↓\nAdmin adiciona questões\n    ↓\nQuestões recebem torneio_id automaticamente ✓\n    ↓\nQuestões aparecem na aba Questões ✓\n```\n\n### Fluxo 2: Torneio → Participantes\n```\nTorneio publicado\n    ↓\nParticipantes se inscrevem\n    ↓\nSistema valida limite máximo\n    ↓\nParticipantes aparecem na aba\n```\n\n### Fluxo 3: Participantes → Ranking\n```\nParticipantes fazem tentativas\n    ↓\nSistema calcula pontuação\n    ↓\nRanking atualiza em tempo real\n    ↓\nCertificados gerados automaticamente\n```\n\n### Fluxo 4: Ranking → Resultados\n```\nRanking finalizado\n    ↓\nResultados exportáveis\n    ↓\nCertificados disponíveis para download\n```\n\n---\n\n## 📈 Endpoints da API em 1 Página\n\n### Torneios\n```\nPOST   /api/tournaments                 → Criar\nGET    /api/tournaments/:id             → Obter\nPUT    /api/tournaments/:id             → Atualizar\nDELETE /api/tournaments/:id             → Deletar\nGET    /api/tournaments/:id/questions   → Questões\nGET    /api/tournaments/:id/participants → Participantes\nGET    /api/tournaments/:id/ranking     → Ranking\nGET    /api/tournaments/:id/statistics  → Estatísticas\nPOST   /api/tournaments/:id/draft       → Salvar rascunho\nGET    /api/tournaments/:id/draft       → Carregar rascunho\n```\n\n### Questões (Integração)\n```\nPOST   /api/questoes                    → Criar com torneio_id\nGET    /api/questoes?torneio_id=:id     → Listar por torneio\nPUT    /api/questoes/:id                → Atualizar\nDELETE /api/questoes/:id                → Deletar\nPOST   /api/questoes/:id/duplicate      → Duplicar\n```\n\n---\n\n## 🧪
