📞 SUPPORT & TROUBLESHOOTING

If you encounter issues:

Problem: Panel still crashes
Solution: 
  1. Open DevTools (F12)
  2. Check Console for errors
  3. Look for error message
  4. Run: fetch('/api/questoes?status_aprovacao=pendente', ...)
  5. Check API response structure
  6. See: GUIA_TESTE_QUESTOES_PENDENTES.md → TROUBLESHOOTING

Problem: Approved questions still not showing
Solution:
  1. Verify database: SELECT * FROM questoes WHERE status_aprovacao='aprovada'
  2. Check if any approved questions exist
  3. In QuestoesColaboradoresTab, verify URL param: limite (not limit)
  4. Open DevTools Network tab
  5. Check request/response
  6. See: FIXES_QUESTOES_PENDENTES_APROVACAO.md

Problem: Need to revert changes
Solution:
  1. git revert [COMMIT_HASH]
  2. Restart both servers
  3. Clear browser cache
  4. System restored to previous state
